# Spreadsheet Rescue Sprint

Public-safe intake surface for a fixed-scope spreadsheet cleanup service.

- Core price: `$75`
- Public service name: `Spreadsheet Rescue Sprint`
- Public legal name use: disallowed
- Payment: private invoice/payment request only after scope fit is confirmed
- Scope: copied/exported CSV, XLSX, or Google Sheet files under about 5,000 rows

Do not include passwords, account access, private keys, payment details, medical details, legal documents, tax records, or regulated financial records in public issues.

Landing page: `index.html`
Fit-check issue template: `.github/ISSUE_TEMPLATE/fit-check.yml`
